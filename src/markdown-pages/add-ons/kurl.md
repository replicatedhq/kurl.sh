## Advanced Install Options

```yaml
apiVersion: cluster.kurl.sh/v1beta1
kind: Installer
spec:
  kurl:
    airgap: true
    proxyAddress: http://10.128.0.70:3128
    additionalNoProxyAddresses:
    - .corporate.internal
    noProxy: false
    licenseURL: https://somecompany.com/license-agreement.txt
    nameserver: 8.8.8.8
```
